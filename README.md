# SON2

Stream Object Notation to:
* JSON
* YAML
* XML
* Properties
<!---
* CVS

-->

## Build
mvn clean install

and next add line

> alias son2="java -jar project_dir/son2/son2-main/target/son2-main-2.8.3-SNAPSHOT.jar"

> alias son2o="son2 o"

> alias son2y="son2 y"

> alias son2x="son2 x"

to file ~/.bashrc
